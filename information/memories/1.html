SHIFT Метод shift() видаляє перший елемент масиву і повертає цей елемент. Цей метод змінює довжину масиву. FOREACH Метод forEach() виконує ітерацію за елементами масиву один за одним для виконання функції. FILTER Метод filter() створює та повертає масив, що містить елементи, що перевіряють фільтр. MAP Метод map() повертає новий масив, що містить усі елементи вихідного масиву, котрому викликається функція. REDUCE Метод reduce() застосовує функцію, яка є «акумулятором» та обробляє кожне значення масиву (зліва направо), щоб зменшити його до одного значення. SLICE Метод slice() повертає масив, що містить копію частини вихідного масиву, частина визначається початковим індексом та кінцевим індексом (за винятком). Початковий масив не буде змінено. CONCAT Метод concat() використовується для об'єднання одного або кількох масивів шляхом їхнього об'єднання. Цей метод не змінює існуючі масиви, він повертає новий масив, який результат операції. JOIN Метод join() створює та повертає новий рядок символів, поєднуючи всі елементи масиву. У конкатенації як роздільник використовується кома або інший рядок, вказаний як аргумент. REVERSE Метод reverse() перевертає масив. Перший елемент масиву стає останнім, а останній елемент масиву стає першим.
